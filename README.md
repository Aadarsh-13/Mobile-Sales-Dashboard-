
# 📱📊 Mobile Sales Dashboard

A powerful, interactive **Power BI dashboard** designed to visualize and analyze mobile phone sales across time, geography, and product lines. This dashboard empowers business users, analysts, and decision-makers with **data-driven insights** on sales trends, product performance, customer satisfaction, and more.

---

## 🚀 Overview

This project provides a deep dive into mobile sales with the help of intuitive visualizations and filters. You can explore:

- 📅 **Monthly, Quarterly, and Yearly trends**
- 🏙️ **Sales by City** (Geographic insights)
- 📈 **Top Brands & Models**
- 💳 **Payment Method Analysis**
- 🧑‍💼 **Customer Ratings**
- 🕵️ **Detailed KPIs** (Total Sales, Quantity, Transactions, and Average Price)

Built using **Power BI Desktop**, it supports interactive data exploration and performance tracking.

## 🔄 Transform Data & Data Model Overview

This Power BI project features a complete **data transformation pipeline** and a **well-structured semantic model** — enabling fast, reliable, and insightful analysis.

### 🧹 Power Query – Data Transformation Layer
- Cleaned raw sales data using **Power Query Editor**
- Applied essential ETL operations like:
  - Column renaming & removal
  - Data type adjustments
  - Filtering & merging tables
  - Normalizing and shaping data

### 🧠 Data Model – Relationship Design
- Structured using a **star schema**: one fact table (sales) with related dimension tables
- Well-defined **one-to-many** and **many-to-one** relationships
- Leverages **DAX** for calculated measures, KPIs, time intelligence, and custom metrics
